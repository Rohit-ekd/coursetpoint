VB Introduction

Visual Basic 6.0 is third generation event driven programming language first released by microsoft in 1991. It evolved from the earlier DOS version called BASIC. where BASIC means Beginners' All-purpose Symbolic Instruction Code.
Since the Microsoft has released many versions of visual basic, from visual basic 1.0 to the final version visual basic 6.0. Visual Basic is a user friendly programming language anyone to developed GUI window application easily.
When using, visual basic the most important skill you need is to be adept at using the development environment. without the integrated tools in the environment, Visual Basic programming would be done much more cumbersome and difficult. Fortunately, visual basic contains many integrated tools to make application development process simpler. This collection of tools makes up the Integrated Development Environment(IDE). I assume that you have installed visual basic 6. This means you have already decided which edition of visual basic you are going to use.

  • The Visual Basic Learning edition
  • The Visual Basic Professional edition
  • The Visual Basic Enterprise edition

The Visual Basic Learning edition is the introductory edition that let you easily create window application

The Visual Basic Professional edition is for computer Professionals and includes advanced features such as tools to develop activeX and Internet controls.

The Visual Basic Enterprise edition is the most advanced edition and is aimed at programmers who build distributed applications in the environment. It includes all the features of the professional edition, plus tool such as Visual sourcesafe and automation.

Integrated development Envirnoment

Visual basic is not just a language. It's an Integrated Development Envirnoment in which you develop, run, test and debug your applications. Start Visual basic and you will see the window as shown figure. This is where you prompted to select the type of project you want to create.

From this window, you can select from several types of projects that will give you a head start on developing your application. This window has three tabs: New, Existing, and Recent.

By selecting a project template from the new tab, you let visual basic create the foundation of your application. This can save lot of time designing an application, especially if you are new to visual basic.
