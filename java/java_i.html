Constructor in Java

We know that uses the dot operator to access the instance variables and then assign values to them individually. It is tadious approch to initialize all the variables of all the objects.
Java supports a special type of method, called a Constructor, that enables an object to initialize itself when it is created.
Constructor have the same name as the class itself. Secondly, they do not specify a return type, not even void. This is because they return the instance of the class itself.
let us consider Point class. and create a constructor method as shown below: 

                            
        // constructor in java
        class Point{
          // declare constructor
          Point(){
             System.out.println("Constructor call");
            }
            public static void main(String [] args){ 
               Point obj = new Point(); //constructor call
             }
         }                    
                        
Output: Constructor call

In the above example demostrate constructor call when we create an object by default it call the constructor of that class.

Rules for creating Java constructor.

There are several rule to create constructor in java:

  1. The constructor name must be the same as the class name.
  2. Constructors do not have a return type, not even 'void'.
  3. If you don't explicitly define any constructors in your class, Java provides a default constructor with no arguments. However, if you define any constructor, including a parameterized one, Java does not provide the default constructor.
  4. You can define constructors with parameters to initialize instance variables with specific values.
  5. Like methods, constructors can be overloaded, meaning a class can have multiple constructors with different parameter lists.
  6. Constructors can call other constructors in the same class using the this() keyword. This is known as constructor chaining.
  7. Constructors are used to initialize the object's state by assigning initial values to its instance variables
Types of Constructor

There are Three types of constructor in java are mentioned below:

Default Constructor

If a class does not explicitly define any constructors, Java provides a default constructor with no arguments. This default constructor initializes instance variables to their default values.

Syntax: 
            Class ClassName{
                //default constructor
                ClassName(){
                    //code
                }
            }
        
                                
         class CTP{
           CTP(){
              System.out.println("default Constructor call");
             }
             public static void main(String [] args){ 
                CTP obj = new CTP(); //constructor call
              }
          }                   
                            
Output: Default constructor
Parameterized Constructor

Parameterized Constructor is similiar to default constructor but parameter list is passed into the constructor method.
A parameterized constructor in Java is a constructor that accepts parameters. It allows you to initialize an object with specific values at the time of creation. 

                                
        class Rectangle{
            int length , width;
           // parameterized Constructor
            Rectangle(int length, int width){
               this.length = length; 
               this.width = width;
              }
             int rectArea(){
                return length * width;
                }
             public static void main(String [] args){
                Rectangle rect = new Rectangle(12,5); //constructor call
                 int area = rect.rectArea();
                 System.out.println("Area of Rectangle : " + area);
               }
           }                       
                            
Output: Area of Rectangle : 60
Copy Constructor

A copy constructor in Java is a special type of constructor that creates a new object by copying the value of another object of the same class. It allow you to create a copy of an existing object, ensuring that the new object has the same state as the original. 

                                
        // Java Program for Copy Constructor
        class CTP {
        // data members of the class.
           String name;
           int age;
        
          // Parameterized Constructor
             CTP(String name, int age)
              {
                  this.name = name;
                  this.age = age;
               }
        
             // Copy Constructor
             CTP(CTP obj2)
             {
                  this.name = obj2.name;
                  this.age = obj2.age;
              }
        }
        class CopyConstrutor {
           public static void main(String[] args)
             {
                  // invoke the parameterized constructor.
                  CTP object1 = new CTP("Aditya", 22);
                 System.out.println("\t Name :  " + object1.name+ " and Age : " +object1.age); 
                 System.out.println("*******************************************");
                 //  invoke the copy constructor.
                CTP object2 = new CTP(object1);
                System.out.println("\t Name : " + object2.name+ " and Age : " + object2.age);
               }
        }                                                   
                            
Output: 
                Name : Aditya and Age : 22
        *******************************************
                Name : Aditya and Age : 22
Constructor Overloading in Java

Constructor Overloading is a approach in which more than one constructor have different parameter list and that perform different tasks. below example provide better understanding about constructor Overloading.

                                
        class Main {
            int value;
           // Parameterized constructor with one integer parameter
            Main(int value) {
               this.value = value;
           }
        
           // Parameterized constructor with two integer parameters
            Main(int x, int y) {
               this.value = x + y;
           }
        
           // Parameterized constructor with a String parameter
            Main(String str) {
               this.value = Integer.parseInt(str);
           }
        
           // Display method
           public void display() {
               System.out.println("Value: " + value);
           }
        
           public static void main(String[] args) {
               // Creating objects using different constructors
               Main obj1 = new Main(10);   // Parameterized constructor with one parameter
               Main obj2 = new Main(5, 7); //  with two parameters
               Main obj3 = new Main("10032");  //  with a String parameter
        
               // Displaying values
               obj1.display(); 
               obj2.display(); 
               obj3.display(); 
           }
        }
         
Output: 
         Value: 10
         Value: 12
         Value: 10032
