ActiveX

A loosely defined set of technologies developed by Microsoft for sharing information among different applications.

ActiveX is a deprecated software framework developed by Microsoft that adapts its earlier Component Object Model(COM) and Object Linking and Embedding(OLE) technologies for content downloaded from a network, particularly from the world wide web.

  • ActiveX Technologies ActiveX components are small program module used primarily in Microsoft's Internet Explorer web browser to add features and functionality. These components can be used for various purposes. There are different technology components.
    • COM: ActiveX is based on COM, Which provide a way for software components to interact with each other.
    • OLE: ActiveX adapts OLE technology for embedding and linking conent.
    • ActiveX Controls: These are reusable software components that can be embedded in applications or web pages.
  • Using ActiveX in Java: In practice, Java and ActiveX Operate in different environments and have different models for handling component and security.

    • Platform Independent
    • Popularity and Support
ActiveX control

An ActiveX control is similar to a Java Applet. Unlike Java Applets, however ActiveX controls have full access to the window Operating System. This gives them much more power than Java Applets, but with this power comes a certain risk that the applet may damage the software or data on your machine.

How do ActiveX controls work?
  • ActiveX's controls are typically used to provide interfaces of data and application.
  • For Example, One of the most common uses is to host a visualization contols that display a document on a webpages, where the document is normally represented in an HTML file.
  • When you want to previewer print the document, you use it from with your browser.
ActiveX Code Component

There are several different type of ActiveX Objects:

  • ActiveX components: An ActiveX Object that can be used by other programs.
  • ActiveX control: A special type of ActiveX camponent that function as a control window.
  • ActiceX Container: A program that can accept activeX control or document Object.
  • ActiveX Server: A DDL or other executable program that provides the services of one or more ActiveX Components.
  • ActiveX Client: A program that users an Active Component
  • ActiveX document: Object a document that can be linked or embedded into an ActiveX container.
Types of ACtiveX Control

There are various types of controls are used in window environments and are often embedded in web pages or used in desktop applications.

  1. UI Controls: These controls provides user interface elements like buttons, text boxes, check boxes etc, with in a web page or application.
  2. Multimedia Controls: These controls are used to play audio and video within a web page or an Application. They include controls for video playback , audio playback.
  3. Network Controls: These controls enable networks communication, such as sending and receiving data over the internet or a local network.
  4. Security Controls: ActiveX control also provide security features, such as encryption and decryption of data and digital signature.
  5. Data Access Controls: These controls provide access to the data from databases or other storage systems.They are used for retrieving, display and update data into databases.
  6. Office Automation Controls: These controls are designed to interact with Microsoft Office Application like Word, Excel, Outlook and more.
ASP(Active Server Page)

History: ASP was firstly introduced in December 1966, as part of IIS(Internet Information Services) 3.0 which was called ASP 1.0. Then it subsequent version came in 1997 and 2000 named ASP 2.0 and ASP 3.0 respectivily.

ASP Stands for Active server Pages and it is a server side script engine for building web pages. ASP basically a server page that contains embedded programs in it. The ASP server side engine basically perform the tasks of reading executing the ASP file and then return the file to the browser.

It is basically a web based framework that help programmer to buid dynamic web pages.
