Access Modifier in Java

In Java, Access modifier help to provide security and accessibility of classes, variables, methods, and constructors. They control access level that other classes and pakages. Access modifier restrict the scope of class, constructor, variable, method.

Default Access Modifier

There is no access modifier is specified, it is considered as default. Default members are accessible within the same class and package.
program to illustrate the working of default access modifier.
In this program we have create a package name MyPackages and class Display try to access method from the package using default access modifier. 

                            
package MyPackages;

// This class has default access modifier 
class Display
{ 
    void show() 
    { 
        System.out.println("Hello Java!"); 
    } 
} 
                            
                        
                            
 /*  program to illustrate error while using
 class from different package with default modifier */
 import MyPackages.Display; 
                         
 class NewClass 
 { 
     public static void main(String args[]) 
     { 
         // Accessing class Display from package MyPackage
         Display obj = new Display(); 
         obj.show(); 
     } 
 } 
                            
                        
Output:
            NewClass.java:3: error: Display is not public in MyPackages; 
            cannot be accessed from outside package
            import MyPackages.Display;
                             ^
Public access Modifier

We can decalare public access modifier using keyword "public". If we can declare any method, class, data members as public it can access everywhere in program. Thier is no restriction of the scope.

Program to demonstrate the working of Public access modifier. consider same example we use in default access modifier but litle change class and method declare as public.

                    
 package MyPackages;
 
 // This class has public access modifier 
 public class Display
 { 
     public void show() 
     { 
         System.out.println("Hello Java!"); 
     } 
 } 
             
                
                
import MyPackages.Display; 
 
class NewClass 
{ 
    public static void main(String args[]) 
    { 
        // Accessing class Display from package MyPackage
        Display obj = new Display(); 
        obj.show(); 
    } 
} 
                
            
Output: Hello Java!
Private Access Modifier

The Private access modifier is declare using the keyword "private". The Method, data member declare as private it accessible only within the class. Mean it is not accessible outside the class and package.

Program to show declare a private method in a class they did not accessible outside the class. 

                    
class A{
    private void display(){
       System.out.println("Private Access Modifier");
       }
    public void show(){
        System.out.println("Public Access Modifier");
      }
   }
 class B {
public static void main(String [] args){
  A obj = new A();
  obj.show();
  obj.display();
 }
}
                    
                
Output: 
        B.java:13: error: display() has private access in A
        obj.display();
            ^
            1 error
Protected Access Modifier

The Protected access modifier is declare using the keyword "protected". The method , data member declare as protected are accessible within the same package, same class and different package subclass.

Program to demonstrate protected access modifier. Consider a package MyPackage and Class A in MyPackage is made public, to access by the class B. The method show in class A is protected and class B is inherited from class A and this protected method is then accessed by creating an object in class B.

                    
package MyPackages;

public class A
{ 
    protected void show()  // method show() declare as Protected
    { 
        System.out.println("Welcome in courseTpoint!"); 
    } 
} 
                    
                
                    
import MyPackages.*; 
class B extends A // class B is Subclass of A
{ 
    public static void main(String args[]) 
    { 
        B obj = new B(); 
        obj.show(); 
    } 
} 
                    
                
Output: Welcome in courseTpoint!
